Featured work

ESP32 + LoRa

Military Pager System

The challenge: Reliable communication without cellular or satellite infrastructure.

My approach: I built a LoRa mesh network with ESP32 nodes, implementing duty-cycled reception for battery life and AES encryption for security. The base station aggregates messages and monitors link quality via RSSI/SNR.

📡 Node ⇄ Base Station (multi‑km LOS)

Technologies: ESP32, 433MHz LoRa, FreeRTOS, AES

Results: Achieved 5km range, 0.5s latency, 6 months battery life on 18650 cells. Successfully tested in remote mountain terrain.

GroqShell ESP32‑S3

The challenge: Bring LLM intelligence to resource-constrained edge devices.

My approach: I created a UART-based shell that connects to Groq's cloud API over TLS, streams prompts, and displays responses. The firmware handles JSON parsing with cJSON and multitasking with FreeRTOS.

[UART] ⇄ [ESP32‑S3] ⇄ [Groq API (LLaMA 3.1)]

Technologies: ESP‑IDF, FreeRTOS, cJSON, TLS 1.2

Results: 200ms average inference latency, interactive AI debugger, <100kB RAM footprint. Open-sourced for embedded ML community.

Cycle‑accurate DSP Engine

The challenge: Real-time filtering on an 8-bit MCU without floating-point hardware.

My approach: I implemented an 8-tap FIR filter using Q15 fixed-point arithmetic, timer-triggered ADC sampling at 8kHz, and hand-optimized assembly loops for cycle counting.

ADC ➔ Q15 ➔ 8‑tap FIR ➔ UART @ 115200

Technologies: ATmega2560, avr-gcc, custom linker script

Results: 78 cycles per tap, 8kHz sampling without missed samples. Filtered data streamed in real-time to Python for analysis.
